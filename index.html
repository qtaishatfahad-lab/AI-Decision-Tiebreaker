Fahad's AI Tiebreaker

A decision-support tool powered by Llama-3 via the Groq API. Enter any dilemma and the AI analyses it across three structured frameworks — Pros & Cons, Comparison Table, and SWOT — before delivering a confident final verdict.

  • Model: Llama-3.1-8B (Groq)
  • Backend: Node.js + Express
  • Frontend: Vue.js 3
  • Security: API key stored server-side
